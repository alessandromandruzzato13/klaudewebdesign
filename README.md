# Hôtel Miramar Monte-Carlo · Équivoque

Site rebuild commissioned via **Klaudewebdesign** (Lapovision Web discipline · service-line).

- **Client**: Nicola Buratto · General Manager Hôtel Miramar Monte-Carlo · Founder Équivoque Rooftop
- **Build by**: Lapo Mandruzzato · klaudewebdesign
- **Stack**: HTML + React 18 (CDN) + Babel standalone + custom CSS · static deploy Vercel
- **Live**: https://miramar-monte-carlo.vercel.app

## Structure

- `index.html` — Miramar entry · loads `app.jsx` + `sections.jsx`
- `equivoque.html` — Équivoque entry · loads `equivoque.jsx`
- `styles.css` — Miramar editorial luxury system
- `equivoque.css` — Équivoque nocturnal perfume system
- `vercel.json` — static deploy config

## Deploy

Vercel CLI:
```
vercel --prod
```

Or GitHub auto-deploy via Vercel integration.
